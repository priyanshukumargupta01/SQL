# SQL
🌱 Learning SQL

//Simple Definition (Exam ke liye) 
> **SQL (Structured Query Language) is a language used to communicate with and manage data in a relational databaseBY DBMS(DataBase Management System).**

//Hindi/Hinglish 
SQL ek language hai jiske through hum database se baat karte hain aur data par operations perform karte hain, jaise:

* Data dekhna (**SELECT**)   //READ
* Data add karna (**INSERT**)   //CREATE
* Data update karna (**UPDATE**)  //UPDATE
* Data delete karna (**DELETE**)   //DELETE

//Example 
```sql
SELECT * FROM Students;
```

Yeh query Students table ka saara data dikha degi.

**Short Definition:**

=> SQL is the language used to manage and manipulate data in a database.
 

> How SQL Works

User
  ↓
SQL Query
  ↓
Database Management System (DBMS)
  ↓
Result Returned
